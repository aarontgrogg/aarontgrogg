Demo for Pure CSS Highlight for Link :target

This is a demo for a pure CSS highlight for link targets.

You might want to read the blog post, but then again, you might not...

What should happen when you click the links below (check browser support for this CSS feature) is that the page should jump down to the link's target, which is much lower on the page, then the targeted section should receive a highlight that will fade-in, then slowly fade-away.
